Troubleshooting

Connection Issues

  • Make sure your Arduino is connected to your computer
  • Check that you have the correct Arduino sketch uploaded
  • Try a different USB cable or port
  • Ensure no other programs are using the serial port

RFID Reader Issues

  • Verify all wiring connections between Arduino and RFID module
  • Check that the RFID module is powered correctly (3.3V)
  • Try placing the card directly on the reader surface
  • Move away from sources of electromagnetic interference

Card Format

Card numbers must be exactly 10 digits for SmartBite system compatibility. The Arduino sketch automatically formats the RFID card UID to match this requirement.
